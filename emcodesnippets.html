Esta guía se refiere únicamente a CSS Buttons, utilizados tipicamente para dar estilos a links especificos. Los estilos han sido tomados del tema base de cada customer y algunos otros de ellos usan librerias de bootstrap. Para mayor información información por favor consulte la guía: Bootstrap Buttons

Todos los botones siguen una estructura de diseño más o menos similar, con algunas variaciones. Con base en esto, se han diseñado dos tipos: boton clásico y botón full width.

Botón Clásico

Típico botón fácilmente utilizable en cualquier página. Se adapta segun el ancho del texto que este dentro de éste.

*modal box - show code*

Botón Full Width - Responsive

Si requiere que el boton ocupe todo el ancho de la página mientas mantiene un comportamiento responsive, solo necesita agregar la clase "fw-btn" al código HTML del botón. El texto será automáticamente centrado.

Add ons

*Agregar sombra al botón.
Si desea agregar una sombra al botón para cambiar su estilo, solo agregue la clase "btn-shadow" al código HTML del botón.
